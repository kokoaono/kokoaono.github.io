What is the difference between Margin, Border, and Padding?

Margin is the space around the content. This determines the space surrounding an element. Margins can be used to move an element's position (left, right, top, bottom) on the webpage.
For example, if you have a piece of land and you want to build houses on it. You would consider where each house will be built and how far away from each other.
The space between houses is called “whitespace”. Whitespace is crucial for creating webpages visually pleasant.

Padding on the other hand, creates space inside an element. An increased value of padding will create more space around the content. This, however, will not change the size of a content. Therefore, it is useful when you want to expand the clickable area such as buttons.

Border is in between padding and margin. This can determine style and colour of an element 's border.
